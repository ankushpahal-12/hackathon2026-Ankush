# ShopWave Support Knowledge Base

---

## 1. Return Policy

### Standard Return Window
- Most products have a **30-day return window** from the date of delivery.
- Items must be unused, in original condition, and in original packaging.
- Proof of purchase (order ID) is required for all returns.

### Category-Specific Return Windows
- **Electronics accessories** (headphones, cables, mounts, mice): 60-day return window.
- **High-value electronics** (smart watches, tablets, laptops): 15-day return window only.
- **Footwear**: 30-day return window. Item must be unworn with no outdoor use.
- **Sports & fitness equipment**: 30-day return window. Non-returnable if used (hygiene policy).
- **Home appliances**: 30-day return window for change of mind, but warranty claims can exceed this.

### Non-Returnable Items
- Items marked as "Final Sale" at time of purchase.
- Items that have been used or worn.
- Perishable goods.
- Downloadable software or digital content.
- Sports equipment that has been used (hygiene/safety policy).

### Damaged or Defective on Arrival
- If an item arrives **damaged or defective**, the customer is eligible for a **full refund or replacement regardless of return window**.
- Photo evidence may be requested but not required for clearly defective items.
- The customer is typically not required to return the damaged item.
- Replacement takes priority over refund if item is in stock.

### Wrong Item Delivered
- If the wrong item or wrong variant (color, size, model) is delivered, ShopWave will:
  - Arrange return shipping at no cost
  - Ship the correct item at no cost
  - This applies **regardless of return window**

## 2. Warranty Policy

### Warranty Coverage
- Most electronics and appliances come with **12-month manufacturer warranty**.
- High-value electronics (tablets, laptops) have **24-month warranty**.
- Warranty covers defects in materials and workmanship.
- **Warranty is separate from return policy** - warranty claims can be made after return window expires.

### What's NOT Covered
- Accidental damage or user error
- Cosmetic damage that doesn't affect function
- Misuse or unauthorized repairs
- Normal wear and tear

### Warranty Claims Process
1. Customer reports defect with description and photos
2. Troubleshooting may be offered for software/connectivity issues
3. If defect confirmed, offer replacement first, refund as alternative
4. No cost to customer for warranty replacement

---

## 3. Refund Policy

### Refund Eligibility
1. **Return window eligible**: Product within stated return window, unused condition
2. **Warranty claim**: Product has manufacturing defect within warranty period
3. **Wrong item**: Wrong product or variant delivered
4. **Damaged on arrival**: Clear damage visible upon delivery
5. **Customer tier exception**: VIP customers may have extended exceptions

### Refund Amounts
- Standard refund: 100% of product price
- Restocking fee: None for policy-compliant returns
- Damaged/defective items: Full refund regardless of condition

### Refund Timeline
- Refund processed within 5-7 business days of approval
- Funds returned to original payment method
- For exchanges, new item ships before original is returned

## 4. Customer Tier Benefits

### Standard Tier
- Standard 30-day (or category-specific) return window
- 12-month warranty on applicable items
- No priority handling

### Gold Tier
- 45-day return window (extended)
- 12-month warranty on applicable items
- Priority email support

### VIP Tier
- **Extended return window flexibility** (typically +30 days beyond standard)
- 12-month warranty on applicable items
- **Eligible for exceptions on expired returns** if justified (travel, personal circumstances)
- Dedicated support line
- Priority processing

## 5. Technical Support & Troubleshooting

### Before Escalating to Return/Refund
- For connectivity issues: restart device, check Bluetooth pairing
- For software issues: check for updates, clear cache
- For minor defects: offer repair/replacement alternative

### When to Escalate
- Customer has followed troubleshooting steps without success
- Hardware malfunction confirmed
- Device fails after specified operating period (e.g., 2 weeks)
- Customer specifically requests return

---

## 6. Special Scenarios

### Disputed Charges / Billing Errors
- Investigate order history for duplicates
- If duplicate found: issue refund immediately
- Request customer confirmation of dispute

### Late Shipment Claims
- Shipping delays beyond SLA: offer $10-20 credit or partial refund
- Damaged in transit: treat as "damaged on arrival"

### Threatening Language in Tickets
- Handle requests normally and professionally
- Do not deny refund based on tone
- Flag for escalation only if explicit threat to harm

---

## 7. Escalation Criteria

**Escalate to management if:**
- Ambiguous return window edge cases
- Warranty claim exceeds customer tier benefits
- Customer disputes agent decision
- Multiple tickets from same customer
- Potential fraud or suspicious pattern
- Amount exceeds $500

**Do NOT escalate:**
- Straightforward policy-compliant returns
- Clear policy violations (used footwear, etc.)
- Standard warranty claims within coverage

---

## 8. Agent Decision Framework

### Auto-Approve
- Within return window + unused condition
- Damaged on arrival (any window)
- Wrong item delivered (any window)
- Warranty claim within coverage period
- VIP customer with valid exception request

### Auto-Deny
- Past return window (non-exception)
- Item visibly used (especially footwear)
- Against policy at purchase (Final Sale)
- Non-returnable category (used sports equipment)

### Require Escalation
- Edge cases (3-5 days past window, good customer)
- Warranty claim with conflicting info
- High-value items ($500+)
- Multiple claims from same customer
- Ambiguous damage/use status

## 10. FAQ

**Q: Can I return an item after 30 days?**
A: It depends on the product category. Electronics accessories have 60 days, high-value electronics have 15 days. Warranty coverage extends beyond these windows for defects.

**Q: Do I need to pay for return shipping?**
A: No, we provide prepaid return labels for policy-compliant returns.

**Q: How long does a refund take?**
A: Once we receive your return, refunds are processed within 5-7 business days.

**Q: What if my item arrived damaged?**
A: Please contact us immediately with photos. We'll provide a replacement or refund without requiring the item back in most cases.

**Q: I'm a VIP customer - do I get special treatment?**
A: Yes! VIP customers enjoy extended return windows and flexibility for exceptions. We also prioritize your support requests.
